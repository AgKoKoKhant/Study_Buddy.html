Study Buddy (Capstone Project)

React App Features

General

  • Register for new account
  • Login to app

Study Groups / Chats

  • Study groups with chatting.
  • Able to send images/videos/files.
  • Able to view images/play videos in full screen by tapping on them from the chat screen.
  • Able to download files sent
  • Search for users to chat 1 to 1
  • Both light mode and dark mode available
  • Mobile responsive for small device
  • Autoscroll to last message when clicking on each chat list.
  • Display message send time based on current timestamp.

Events

  • View all available events with Calendar View
  • Join events or View Discussion (already member of group) by clicking each event
  • Create new events based on Date and Time
  • Have Agenda View.

Books

  • View books recommended based on interests & groups joined
  • Search for books with autocomplete feature
  • View books

Profile

  • View current profile
  • Change profile photo by clicking upload photo (Drag and Drop supported)
  • Edit interests, email, display name
  • Change password
  • Logout

Technical Challenges

React Hook

  • React hook is a bit tricky. If you don’t handle API requests properly using hook effect, you will get memory heat problems.
  • React hook is a newly introduced feature, it takes a bit of time to understand and use it.

Client Site Routing

  • react-router-dom V6 is used for client site routing. It takes a bit of time to know how to use the library.

CORS issue

  • This issue always happens when server and client site are not in the same domain. So, server rejects every request from client because of CORS issue.
  • Able to solve by setting up CORS compatibility setting in backend code.

Styling issue

  • In order to support multi mode in chat style, style was created using SASS instead of CSS. By using SASS, eliminated lots of duplicate code.

HTTPS issue

  • In order to send data over the internet, browser complains unless it is being sent over a HTTPS tunnel. So React is running behind a Nginx reverse proxy providing HTTPS support.
  • If React is running with HTTPS, it can't send data to JAVA backend as it was running without HTTPS. So now all the servers are behind Nginx reverse proxies.

Auto Scroll

  • Auto scroll chat history when there is a new message especially image in React. When a message arrives, auto scroll will be triggered but the image is still loading, and the browser doesn’t know the exact height to be scrolled yet.
  • So able to solve the issue by adding an auto scroll function inside the onload event of the image.

State Management

  • React UI state management. To maintain a single source of truth between Parent and Child components.
